# renderer

a simple point renderer

## Running

to run the code run the following commands

```bash
git clone https://github.com/ogwen123/point-renderer.git
cd point-renderer
git clone https://github.com/libsdl-org/SDL.git vendored/SDL
mkdir build
cd build
cmake ..
cmake --build .
```
